Data Source

The data was collected from kaggle. However, it has been collected and published by the World Health Organization(WHO). It provides information for countries all over the world. For each country it gives data for the years 1979-2016 (some years are missing for some countries). The data consists of the number of suicides and the population size for different age groups. The smallest age group is 5 - 14 and the biggest one is 75+. It also provides information about gender. It is a large dataset consisting of 43776 entries. Preprocessing of the data included removing empty columns and removing information about a year if the number of suicde for all the age groups are 0.
